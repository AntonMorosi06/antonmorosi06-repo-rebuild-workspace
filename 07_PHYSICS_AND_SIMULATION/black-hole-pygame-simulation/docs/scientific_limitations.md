# Scientific limitations

This project is not a real astrophysical simulation.

It does not solve Einstein field equations.

It does not compute relativistic geodesics.

It does not perform gravitational lensing ray tracing.

It does not model magnetohydrodynamic plasma.

It does not model Hawking radiation accurately.

It does not use physical units.

It is a simplified educational and visual model inspired by black hole imagery and orbital dynamics.

This distinction should remain explicit in the README and portfolio description.
