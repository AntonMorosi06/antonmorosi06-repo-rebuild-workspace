# Portfolio summary

Particle World Canvas Lab is a static browser-based particle simulation. It demonstrates vector math, force-field design, Canvas rendering, runtime controls, mouse interaction, presets and responsive UI.

The strongest portfolio angle is that the project can be explained both visually and architecturally. It is not a random animation: it has particles, forces, presets, telemetry and structured documentation.

The project can be connected to MicroBot as a future swarm-state visualizer where particles represent agents and fields represent goals, obstacles, energy gradients or communication zones.
