# Issue 001 - Particle performance

## Goal

Keep the simulation smooth at high particle counts.

## Current implementation

The project uses Canvas 2D and a direct particle loop.

## Future improvements

Add performance mode.

Add FPS counter.

Add adaptive particle limit.

Add spatial partitioning if collision behavior is introduced.

Add WebGL version later.

## Acceptance criteria

The simulation should remain usable on common laptops and browsers.
