# Physics model

The simulation uses a simplified attraction model. Particles are pulled toward a central point by an inverse-square-like force with softening near the center.

A perpendicular swirl term is added so that many particles form curved trajectories rather than falling directly inward. This produces an accretion-disk-like visual structure.

Particles crossing the absorption radius are removed and produce shockwaves. The system respawns particles to maintain a stable visual field.

This model is intentionally educational and artistic. It is not a real relativistic black hole simulation.
