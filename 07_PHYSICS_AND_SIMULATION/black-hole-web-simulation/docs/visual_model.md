# Visual model

The visual system is composed of several layers.

The background contains a procedural starfield.

The accretion glow uses radial gradients around the center.

Particles change color from blue to orange as their energy increases.

Trails show previous particle positions.

Shockwaves appear when particles are absorbed.

Polar jets are drawn as translucent beams.

Lensing rings are stylized ellipses that suggest field distortion.

All visual layers are symbolic and explanatory, not physically exact.
