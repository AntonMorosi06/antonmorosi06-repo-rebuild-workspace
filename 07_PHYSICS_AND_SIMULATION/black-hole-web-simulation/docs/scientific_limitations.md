# Scientific limitations

This project is not a physically accurate astrophysical simulator.

It does not solve Einstein field equations.

It does not trace relativistic geodesics.

It does not compute real gravitational lensing.

It does not simulate magnetohydrodynamic plasma.

It does not model Hawking radiation.

It does not use physical units.

It is a simplified educational and visual model inspired by black hole imagery and orbital dynamics.
