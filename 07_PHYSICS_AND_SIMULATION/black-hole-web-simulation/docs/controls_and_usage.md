# Controls and usage

Run locally with:

python3 -m http.server 8080

Open:

http://127.0.0.1:8080

Controls:

Space pauses or resumes.

R resets.

B creates a burst.

J toggles jets.

T toggles trails.

D toggles debug overlay.

1 selects edge mode.

2 selects disk mode.

3 selects spiral mode.

4 selects rain mode.

5 selects cluster mode.

The interface also includes buttons and sliders for runtime parameter tuning.
