# Portfolio summary

Black Hole Web Simulation is a static Canvas-based physics visualization project. It demonstrates particle systems, vector math, runtime controls, responsive UI, scientific communication and static deployment readiness.

The strongest portfolio angle is that the project looks visually advanced while remaining simple to run. It requires no build process and no external libraries.

It can be presented alongside the Pygame black hole simulation as a web counterpart and as part of a broader physics/visualization portfolio.
