# Controls

Mouse drag:
Rotate the camera around the 3D point cloud.

Mouse wheel:
Zoom in and out.

SPACE:
Show or hide PCA vectors.

R:
Regenerate a new correlated dataset.

C:
Reset camera.

S:
Save screenshot to output.

1:
Fixed point color.

2:
Depth-based color.

3:
Quadrant-based color.

ESC:
Exit.
