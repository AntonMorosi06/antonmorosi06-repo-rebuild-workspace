# Changelog

## v0.1.0 - Reconstructed spectral clustering Pygame baseline

Created a clean skeleton for the Spectral Clustering Pygame Visualizer inside the AntonMorosi06 repository rebuild workspace.

Added a NumPy-based spectral clustering implementation.

Added k-nearest-neighbor affinity graph construction.

Added graph symmetrization.

Added normalized graph Laplacian computation.

Added eigenvector embedding and built-in k-means clustering.

Added Pygame visual interface with interactive point editing, cluster count tuning, neighbor tuning, affinity sigma tuning, graph edge display, screenshot export, help overlay and dirty-state recomputation.

Resolved the original design issue where the S key could conflict between screenshot and zoom behavior by assigning S only to screenshot and using no S-based zoom command.

Added documentation, issues, labels, requirements, tests and repository hygiene files.

No clone from AntonMorosi2234 was attempted.
