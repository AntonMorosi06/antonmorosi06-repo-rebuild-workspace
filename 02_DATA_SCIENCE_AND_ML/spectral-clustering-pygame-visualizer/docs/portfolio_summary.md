# Portfolio summary

Spectral Clustering Pygame Visualizer is an educational machine learning and graph visualization project. It demonstrates how graph affinity and Laplacian eigenvectors can reveal cluster structure that is not obvious through simple centroid-based methods.

The project shows practical understanding of NumPy, Pygame, graph construction, spectral clustering, eigen-decomposition, interactive visualization, repository structure and technical documentation.

The strongest portfolio angle is the combination of mathematical explanation and direct visual interaction. A viewer can change graph neighbors, cluster count and affinity sigma to observe how the graph structure changes the final clustering.

This project can also be connected conceptually to MicroBot telemetry. In a swarm robotics context, a graph can represent communication links, local physical proximity, sensor similarity or behavioral affinity between nodes.
