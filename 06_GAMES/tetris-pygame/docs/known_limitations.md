# Known limitations

This is a Tetris-style educational game, not an official Tetris implementation.

The rotation system is SRS-inspired but incomplete.

There is no T-spin detection.

There is no lock delay system.

There is no advanced DAS/ARR movement tuning.

There is no music or sound system.

The high score system stores only one local score.

The project currently uses generated shapes and no external visual assets.
