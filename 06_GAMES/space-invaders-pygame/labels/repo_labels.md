# Suggested labels

documentation - README, docs and usage guides.

enhancement - Improvements to gameplay, UI or structure.

game-core - Core movement, collision, score and waves.

pygame - Rendering and input interface.

arcade - Arcade shooter behavior and balancing.

assets - Sprites, sound and future visual polish.

highscore - Local runtime persistence.

portfolio - Public-facing presentation improvements.

reconstructed-skeleton - Files rebuilt cleanly without cloning the old repository.
