# Issue 001 - Core gameplay balance

## Goal

Improve the feel and balance of the arcade loop.

## Current implementation

Aliens move horizontally, drop when they hit the edge, fire randomly and respawn in waves after being cleared.

## Future improvements

Add shields.

Add alien types with different scores.

Add difficulty presets.

Tune alien speed and fire chance.

Add explosion feedback.

Add player invulnerability frames after being hit.

## Acceptance criteria

The game should be playable for several waves without becoming instantly unfair.
