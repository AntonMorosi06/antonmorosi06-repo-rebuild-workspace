# Controls and usage

Text mode:

python3 main.py --mode text

Tkinter mode:

python3 main.py --mode tkinter

Curses mode:

python3 main.py --mode curses

Pygame mode:

python3 main.py --mode pygame

General controls:

Arrow keys move.

WASD also move in supported modes.

R resets in curses, Tkinter and Pygame modes.

Space pauses in Pygame mode.

Q or Escape quits where supported.
