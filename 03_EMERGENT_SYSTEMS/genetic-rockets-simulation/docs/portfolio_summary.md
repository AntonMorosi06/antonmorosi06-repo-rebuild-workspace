# Portfolio summary

Genetic Rockets Simulation is an educational Pygame project that demonstrates a genetic algorithm through movement, collision and target-seeking behavior.

The project shows practical understanding of Pygame, object-oriented Python, simulation loops, vector motion, fitness functions, selection, crossover, mutation, documentation and repository organization.

The strongest portfolio angle is the visual emergence of better behavior over generations. The first rockets move almost randomly. After repeated selection and mutation, the population begins to discover trajectories that avoid the obstacle and approach the target.

This project can also be connected conceptually to MicroBot simulation. The same evolutionary framework could later be used to optimize navigation paths, docking behavior, formation movement or simulated swarm strategies.
