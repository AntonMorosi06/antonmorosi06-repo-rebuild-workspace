# Controls and usage

Run the project with:

python3 main.py

The simulation opens a Pygame window with the rocket canvas on the left and the dashboard on the right.

Space pauses or resumes the simulation.

R resets the population and generation counter.

O creates a random obstacle and resets the population.

C restores the default obstacle and resets the population.

Up increases the mutation rate.

Down decreases the mutation rate.

S saves a screenshot into the screenshots directory.

H toggles the help overlay.

Escape exits the application.

The dashboard shows the current generation, current step, population size, mutation rate, best fitness, average fitness, completed rockets, crashed rockets and best distance from the previous evaluated generation.
