# Suggested labels

documentation - README, guides, screenshots and explanations.

enhancement - Improvements to simulation, controls or structure.

algorithm - Genetic algorithm logic, fitness, selection, crossover and mutation.

simulation - Pygame runtime behavior, obstacles, target and rocket dynamics.

performance - Population size, frame rate and update loop scalability.

portfolio - Public-facing presentation improvements.

microbot-link - Conceptual links to MicroBot simulation, navigation or swarm optimization.

reconstructed-skeleton - Files rebuilt cleanly without cloning the old repository.
