# Evolution model

Evolution in this project is simulated through differential survival and reproduction.

There is no explicit fitness function assigned from outside. Instead, fitness emerges indirectly from the environment. Agents that find food, manage energy costs and reproduce successfully leave more descendants.

Traits can have trade-offs.

High speed can help an agent reach food quickly, but it increases energy cost.

High perception can help an agent detect food, but it also has a cost.

High fertility can increase reproductive output, but reproduction can become dangerous when food is scarce.

High mutation intensity can explore new traits, but it can also destabilize successful lineages.

The model is intentionally simplified and should be used for education and visualization, not biological prediction.
