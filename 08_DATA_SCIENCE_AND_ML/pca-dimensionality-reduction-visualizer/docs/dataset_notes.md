# Dataset notes

The datasets are synthetic and generated locally.

Correlated cloud contains two variables that share a strong common direction.

Ellipse contains an elongated rotated distribution.

Rotated clusters contains multiple clusters arranged along a dominant direction.

3D ribbon contains three-dimensional samples that can be reduced into a two-dimensional principal-component view.

Noisy line contains an approximate linear relation with noise.

These datasets are designed to show why PCA can reveal dominant variance directions.
