# Known limitations

The PCA implementation is educational and not optimized for very large matrices.

The Jacobi eigen decomposition is suitable for the small covariance matrices used in this project.

There is no CSV import yet.

There is no animated step-by-step mode yet.

There is no 3D interactive renderer yet.

The Pygame interface is a visual learning tool, not a full data analysis platform.

For production-scale PCA, optimized numerical libraries should be used.
