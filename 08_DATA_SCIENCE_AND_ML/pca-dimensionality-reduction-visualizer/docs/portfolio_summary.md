# Portfolio summary

PCA Dimensionality Reduction Visualizer is an educational machine learning visualization project. It demonstrates centering, covariance, eigen decomposition, principal components, explained variance, projection and reconstruction.

The project shows practical understanding of mathematical programming, pure Python linear algebra, synthetic dataset generation, Pygame rendering, visual explanation and tests.

The strongest portfolio angle is that PCA is implemented directly instead of hidden behind a library call. This makes the repository useful for explaining the algorithm during interviews, study sessions or project reviews.
