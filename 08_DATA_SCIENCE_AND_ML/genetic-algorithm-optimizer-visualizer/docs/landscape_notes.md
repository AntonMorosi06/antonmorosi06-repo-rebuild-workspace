# Landscape notes

Sphere is a simple convex bowl. It is easy for the genetic algorithm because there is only one global optimum.

Rastrigin is multi-modal. It has many local optima, so mutation and diversity matter.

Himmelblau has several equivalent optima. The population may converge toward different good regions.

Ridge rewards following a curved structure. It shows how populations can align along a fitness path.

Multi-peak contains several local peaks. It is useful for showing premature convergence and the value of mutation.
