# GitHub Pages checklist

The dashboard is static and can be deployed through GitHub Pages.

Checklist:

Generate web/data/sample_report.json.

Open web/index.html locally.

Serve the web directory with python3 -m http.server 8080.

Check that metric cards render.

Check that column cards render.

Check that correlations render.

Check browser console for errors.

Add screenshots before public presentation.
