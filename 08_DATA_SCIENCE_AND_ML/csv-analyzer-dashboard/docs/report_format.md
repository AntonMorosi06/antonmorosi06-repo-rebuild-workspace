# Report format

The JSON report contains source, row_count, column_count, columns and correlations.

Each column profile contains name, inferred_type, total_count, missing_count, missing_ratio, unique_count, numeric_summary and top_values.

Numeric summaries contain count, min, max, mean, median, standard deviation, first quartile and third quartile.

Correlations contain column_a, column_b, correlation and pair_count.

The Markdown report is designed for quick reading in GitHub or project documentation.
