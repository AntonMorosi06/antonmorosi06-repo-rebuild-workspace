# Portfolio summary

Spectral Clustering Visualizer is an educational machine learning visualization project. It demonstrates graph construction, similarity kernels, normalized Laplacian, eigenvectors, spectral embedding and final k-means clustering.

The project shows practical understanding of graph-based machine learning, linear algebra, clustering, synthetic dataset generation, Pygame rendering, documentation and tests.

The strongest portfolio angle is that the pipeline is implemented directly instead of being hidden behind a library call. This makes the project useful for explaining spectral clustering in interviews, study sessions or technical reviews.
