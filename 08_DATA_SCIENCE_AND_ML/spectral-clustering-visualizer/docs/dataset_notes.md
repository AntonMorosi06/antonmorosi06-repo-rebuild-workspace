# Dataset notes

The datasets are synthetic and generated locally.

Moons contain two curved half-moon shapes.

Rings contain two concentric circular clusters.

Blobs contain compact Gaussian groups.

Bridge contains two dense regions connected by a sparse bridge.

Islands contain several compact disconnected groups.

Spiral contains two interleaved spiral-like arcs.

These datasets are selected because spectral clustering is especially useful when Euclidean shape alone is misleading.
