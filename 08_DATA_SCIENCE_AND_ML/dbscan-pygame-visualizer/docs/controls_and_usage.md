# Controls and usage

Run with:

python3 main.py

Controls:

R reclusters the current dataset.

B loads blobs.

M loads moons.

O loads rings.

N loads random noise.

X loads mixed dataset.

Up increases epsilon.

Down decreases epsilon.

Right increases min_samples.

Left decreases min_samples.

Tab selects the next point.

C toggles core point overlay.

H toggles selected-point neighborhood.

D toggles debug footer.

S saves a screenshot.

Q or Escape exits.
