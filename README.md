# AntonMorosi06 Repo Rebuild Workspace

This workspace is a local staging area for rebuilding, cleaning, documenting and preparing selected repositories originally analyzed from the AntonMorosi2234 GitHub account.

The goal is not to copy everything blindly. The goal is to create improved, portfolio-ready, technically honest versions of the repositories, with better structure, clearer README files, issue backlogs, labels, documentation, limitations, screenshots, and future GitHub publication plans.

This workspace is designed to be pushed later to AntonMorosi06 as a master rebuild repository, or used as a staging area before splitting selected projects into independent repositories.

Important rules:

- Original repositories are not modified by this script.
- No source code is imported automatically.
- Each project folder contains a planned improved structure.
- Every project has an audit file, issue backlog, labels file and publication notes.
- Some projects are marked as portfolio-ready candidates.
- Some projects are marked as study archive or recovery placeholder.
- Claims must remain technically honest.
- Hardware, AI, healthcare and privacy-related projects must include limitations and safety notes.
