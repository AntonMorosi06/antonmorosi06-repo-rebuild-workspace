# QEMU Validation Checklist

- [ ] Run `make clean`.
- [ ] Run `make`.
- [ ] Confirm `build/microbot_os.img` exists.
- [ ] Run `make run`.
- [ ] Confirm QEMU boots.
- [ ] Confirm home screen appears.
- [ ] Press 1, 2, 3, 4, 5.
- [ ] Confirm screen switching.
- [ ] Save screenshot in `demos/screenshots`.
- [ ] Save terminal output in `demos/logs`.
