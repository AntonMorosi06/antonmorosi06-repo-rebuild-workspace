# Browser test checklist

Tested items to verify manually:

The page loads without console errors.

Synthetic mode starts automatically.

The tesseract projection rotates.

The spectrum bars move.

The particle field reacts to synthetic energy.

The sliders update the visualization.

Start microphone asks for permission.

Stop microphone returns to synthetic mode.

The layout remains usable on a narrow screen.

The conceptual note is visible in the control panel.

The GLB placeholder is visible for future expansion.
