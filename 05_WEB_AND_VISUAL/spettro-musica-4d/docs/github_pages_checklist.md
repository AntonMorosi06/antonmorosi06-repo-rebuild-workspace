# GitHub Pages checklist

This project is a static site and can be deployed with GitHub Pages.

Recommended checklist:

Confirm index.html opens locally.

Serve the folder with python3 -m http.server 8080 and test in the browser.

Check that synthetic mode works without microphone permission.

Check that microphone mode works on localhost.

Push the repository to GitHub.

Enable GitHub Pages from the repository settings if this folder is published as a standalone repository.

Use HTTPS when testing microphone mode online.

Add screenshots to the screenshots directory.

Add a short demo GIF or video later if needed.

Keep the conceptual note visible to avoid presenting the 4D visualization as a physical proof.
