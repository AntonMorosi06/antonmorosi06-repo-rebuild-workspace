# Conceptual model

Spettro Musica 4D uses the fourth dimension as a mathematical and informational visualization device.

A tesseract can be represented as a set of points with four coordinates: x, y, z and w. Since a normal screen is two-dimensional, the system rotates those four-dimensional points and projects them down into screen coordinates.

The audio signal does not open a physical fourth dimension. It modulates the visualization. Energy, frequency bands and synthetic or microphone-derived audio values influence scale, color, particle displacement and apparent depth.

This distinction is important. The project is suitable as a creative coding and educational dimensional visualization, but it must not be described as experimental evidence of real 4D physics.

In a MicroBot context, the same logic can be reused more rigorously as a dimensional state engine: x, y and z may represent spatial coordinates, while w may represent energy, phase, confidence, communication state, time offset or another telemetry dimension.
