# Accessibility notes

The page includes a skip link for keyboard navigation.

Navigation links use normal anchors.

Buttons are real button elements.

The canvas includes an aria-label, but the technical information is also available in text form through the subsystem panel.

Color is not the only way information is communicated. The selected subsystem also updates title, description, code and theme.

Future accessibility improvements should include keyboard shortcuts for subsystem selection, reduced-motion mode, stronger contrast testing and a text-only technical summary.
