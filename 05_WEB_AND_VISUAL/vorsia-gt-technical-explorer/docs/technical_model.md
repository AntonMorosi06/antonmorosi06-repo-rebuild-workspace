# Technical model

The technical model is intentionally fictional but structured.

The vehicle is described as a hybrid GT concept. The main systems are hybrid drivetrain, active aerodynamics, carbon chassis, brake and thermal system and telemetry cockpit.

The canvas view uses a stylized side silhouette rather than a CAD model. Subsystem zones are approximate and exist only for visual explanation.

The telemetry dashboard is synthetic. It does not represent real vehicle data. Speed, battery assist, brake temperature and aero load are generated from deterministic oscillating values so that the interface feels alive during a demo.

Future versions can replace the canvas silhouette with a real GLB model, add a 3D viewer, expose more subsystem layers and include a richer telemetry simulator.
