Aerodynamic model

Active aero surfaces

The fictional model includes front splitter loading, underfloor channel acceleration, rear diffuser extraction and adaptive rear wing angle. These zones are represented visually for technical storytelling and UI demonstration.

Drivetrain model

Hybrid torque vectoring

The concept separates combustion rear drive from electric front axle support. This allows the interface to explain launch assist, corner exit torque shaping and regenerative braking.

Cockpit model

Driver information layer

The dashboard is imagined as a telemetry-first interface: speed, energy assist, brake state, aero load and system warnings are prioritized over decorative information.
