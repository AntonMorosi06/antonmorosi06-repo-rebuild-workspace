# Vorsia GT Technical Explorer

This repository is a clean reconstructed skeleton of the Vorsia GT technical explorer project. It is not a clone of the original repository. It is a rebuilt, documented and improved version prepared inside the AntonMorosi06 repository rebuild workspace.

Vorsia GT is treated here as a fictional technical automotive concept used for portfolio, interface design, visual engineering explanation and static web development. The project is not affiliated with any real automotive manufacturer. It does not claim that the vehicle exists physically. It is a concept explorer that presents a high-performance GT car through technical cards, telemetry panels, aerodynamic zones, drivetrain architecture, chassis systems, cockpit instrumentation and an interactive canvas visualization.

## Features

The site includes a hero section, technical specification cards, telemetry dashboard, interactive canvas vehicle silhouette, subsystem explorer, aerodynamic map, drivetrain explanation, cockpit notes, accessibility-aware controls and responsive layout.

The interface is designed to look like a technical configurator or engineering showcase rather than a generic landing page. The canvas view is lightweight and does not require external 3D libraries. It draws a stylized GT silhouette and overlays active subsystem zones.
