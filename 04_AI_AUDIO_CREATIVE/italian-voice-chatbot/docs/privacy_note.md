# Privacy note

The default mode is text-only and does not use the microphone.

The optional voice mode uses microphone input through SpeechRecognition and voice output through pyttsx3. Depending on the recognition backend used by SpeechRecognition, audio recognition may rely on external services. For safe demonstrations, avoid speaking private information, credentials, personal addresses, financial data, health information or sensitive project details.

The reconstructed baseline does not intentionally send transcripts to a custom server. Transcripts are written locally into the transcripts directory when the app saves them.

For portfolio demos, use neutral example conversations.
