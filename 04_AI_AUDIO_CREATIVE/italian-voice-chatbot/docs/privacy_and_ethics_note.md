# Privacy and Ethics Note

This project must include privacy and ethical limitations before publication.

Required checks:

- No API key is committed.
- No .env file is committed.
- No private user data is committed.
- If microphone input is used, explain what is processed locally and what may be sent to external services.
- If OpenAI, Google Speech Recognition, gTTS or other external services are used, state it clearly.
- If the dataset is synthetic, say so clearly.
- If the model makes predictions about people, state that it is educational and not for real decisions.
- If healthcare data or healthcare predictions are involved, state that the project is not clinical software.
