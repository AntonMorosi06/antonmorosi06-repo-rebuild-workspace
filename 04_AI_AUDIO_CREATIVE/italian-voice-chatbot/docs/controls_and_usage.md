# Controls and usage

Run text mode with:

python3 main.py --mode text

Run optional voice mode with:

python3 main.py --mode voice

In text mode, type a message and press Enter.

Use /help to display help.

Use /save to save the transcript manually.

Use esci, exit, quit, chiudi or ciao to close the conversation.

In voice mode, the assistant listens once per turn, replies in the terminal and attempts to speak the answer aloud. If speech recognition or microphone access fails, the app prints an error and recommends text mode.
