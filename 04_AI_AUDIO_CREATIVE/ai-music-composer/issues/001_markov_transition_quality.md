# Issue 001 - Markov transition quality

## Goal

Improve melodic quality while keeping the model understandable.

## Current implementation

The melody model uses weighted transitions over scale degrees.

## Future improvements

Add motif repetition.

Add phrase endings.

Add call-and-response structure.

Add style-specific transition maps.

Add optional training from small symbolic examples.

## Acceptance criteria

Generated melodies should feel less random while remaining reproducible by seed.
