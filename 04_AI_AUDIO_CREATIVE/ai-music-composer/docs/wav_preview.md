# WAV preview

The WAV preview is a simple audible rendering using the Python standard library.

It converts MIDI notes to sine waves, mixes all melodic events into a mono audio buffer and adds a basic synthetic drum noise layer.

This is not a professional synthesizer. It does not use real instruments, samples, effects or mastering. It exists only to let the user quickly hear the generated idea without opening a MIDI editor.

For serious audio production, the MIDI file should be imported into a DAW or played with a better soundfont.
