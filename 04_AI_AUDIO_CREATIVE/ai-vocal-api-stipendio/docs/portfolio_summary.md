# Portfolio summary

AI Vocal API Stipendio is an educational machine learning API project. It demonstrates how to package a synthetic classifier into a CLI and FastAPI service with model persistence, validation, documentation and tests.

The project shows practical understanding of NumPy, binary classification, feature scaling, JSON persistence, FastAPI, Pydantic schemas, command-line interfaces, API documentation, model cards and privacy notes.

The strongest portfolio angle is the full application structure. It is not only a script: it has a dataset layer, preprocessing layer, model layer, persistence layer, service layer, API layer, CLI layer and optional voice/TTS helper.

The project should be presented as an educational ML/API engineering exercise, not as a real salary predictor.
