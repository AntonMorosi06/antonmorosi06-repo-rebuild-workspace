# Dataset note

The dataset is synthetic. It is generated inside the repository and does not represent real people.

The input fields are age, experience, education_level and current_salary. The target label is generated from an artificial mathematical rule with random noise.

The dataset exists only to make the API and ML workflow self-contained. It is useful for learning how to structure a small model-serving project, but it has no real-world validity.

Do not use this dataset or model for hiring, compensation, credit, eligibility, ranking, profiling or any decision affecting real people.
