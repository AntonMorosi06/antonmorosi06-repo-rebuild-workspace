# Model card

## Model name

AI GUI Keras Classifier reconstructed baseline.

## Intended use

The intended use is educational. The model demonstrates a small classification workflow with synthetic data, feature scaling, optional Keras training, NumPy fallback training and GUI prediction.

## Not intended use

The model must not be used for real hiring, salary, financial, educational, medical, legal or eligibility decisions.

## Input features

Age, salary and experience.

## Output

The model outputs a binary synthetic class and a confidence score.

## Data

The data is generated by a synthetic rule. The data is not collected from real people.

## Training backends

The preferred backend is Keras when TensorFlow is installed. The fallback backend is a NumPy logistic classifier.

## Limitations

The model learns an artificial pattern. Its accuracy only reflects how well it matches the synthetic labels. It has no real-world validity.

## Ethical note

Even toy models can encourage harmful habits if their limitations are not explicit. This repository therefore includes a model card and dataset note by default.
