# Ethics and limitations

This project is a toy educational classifier. It must not be used to evaluate real people.

The chosen input fields can resemble sensitive socioeconomic decision contexts. For that reason, the repository clearly separates demonstration from real-world deployment. The synthetic target class does not represent job suitability, creditworthiness, competence, salary fairness or any serious concept.

The correct interpretation is technical: the project shows how to connect a dataset, preprocessing, a classifier and a GUI. It does not show how to make valid human decisions.

Future versions should continue to include explicit warnings, dataset notes and model cards.
