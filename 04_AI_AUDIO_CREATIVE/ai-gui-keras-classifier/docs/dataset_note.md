# Dataset note

The dataset used in this project is synthetic. It is generated inside the repository and does not represent real people.

Each row contains three input features: age, salary and experience. The target label is produced by an artificial mathematical rule with added noise. The rule is designed only to create a learnable classification pattern for educational purposes.

This means the model can be useful for learning how a GUI-based classifier is structured, but it must not be interpreted as a real decision model. It should not be used for hiring, compensation, finance, eligibility, profiling or any real-world decision affecting a person.

The synthetic dataset exists to make the project self-contained, reproducible and safe to run without external private data.
